# 湖湘文脉 · 文遗风华

湖南文化遗产展示柜 —— 沉浸式帝王卷轴开幕 + 水墨山水背景 + 分类/城市筛选轮播

## 功能

- 20个湖南文化遗产条目：10个遗址/物质文化遗产 + 10个非物质文化遗产
- 分类筛选（遗址 / 非遗）与城市筛选，可叠加使用
- 帝王卷轴开幕动画（紫檀轴 + 朱红镶边 + 金光桃花源特效）
- 精选条目轮播
- 详情页（新窗口）
- CRUD 条目管理（支持分类字段）
- 多层水墨山水背景 + 传统纹样 + 粒子动画
- 数据本地存储（localStorage）+ 跨窗口同步
- 响应式布局

## 内容

### 遗址 / 物质文化遗产
岳麓书院、岳阳楼、凤凰古城、张家界武陵源、韶山毛泽东故居、南岳衡山、马王堆汉墓、橘子洲头、洞庭湖、湖南省博物馆

### 非物质文化遗产
湘绣、花鼓戏、土家族织锦技艺、苗族银饰锻制技艺、醴陵釉下五彩瓷烧制技艺、浏阳花炮制作技艺、湖南皮影戏、女书习俗、安化黑茶制作技艺、汨罗江畔端午习俗

## 使用

直接在浏览器打开 `index.html` 即可。
